## 第１章：線形代数
前半で行列の基本的な演算について学び、行列版の逆数である逆行列の求め方（行列式が０となる行列については存在しない）も学んだ。
<br> 後半は、固有値・固有ベクトルの計算方法について学んだ。
<br> 特に、正方行列以外の行列についての固有値分解（特異値分解）については個人的に初めて知り、画像について応用例があることを学んだ。
